    CSS overview

  • CSS refers to Cascading Style Sheets.

The style solves a common problem

HTML tags were originally designed to define the content of the document.By using labels such as <h1>,<p> and <table>, the original intention of the HTML was to convey "This is the title", "This is the paragraph," "This is the table," and so on.

As both major browsers (Netscape and Internet Explorer) continually add new HTML tags and attributes (such as font tags and color attributes) to the HTML specification, creating sites that clearly document content independent of the presentation layer of the document is more and more difficult.

To address this issue, the World Wide Web Consortium (W3C), the non-profit, standardized consortium, took on the task of HTML standardization and created a style outside HTML 4.0.

All major browsers support Cascading Style Sheets.

Style Sheets Greatly Improve Work Efficiency

Style sheets define how to display HTML elements, just as HTML 3.2's font labels and color attributes do.

The style is usually saved in an external .css file. By simply editing a simple CSS document, external stylesheets give you the ability to change the layout and appearance of all the pages in your webpage at the same time.


With the ability to control the style and layout of multiple pages at the same time, CSS is a breakthrough in WEB design.

As a website developer, you can define a style for each HTML element and apply it to as many pages as you like.

For a global update, simply change the style and then all the elements in the website are automatically updated.

Multiple Styles Will Be Casacaded One

Style sheets allow specification of style information in a variety of ways.

Styles can be specified in a single HTML element, in the header element of an HTML page, or in an external CSS file.

Even multiple external style sheets can be referenced inside a single HTML document.

Cascading Order

Which Style Will Be Used When the Same HTML Element is Defined by More Than One Style?

In general, all styles are stacked in a new virtual style sheet based on the following rules, with number 4 having the highest priority.

  1. CSS refers to Cascading Style Sheets.
  2. External style sheet.
  3. Internal style sheet (inside tag)
  4. Inline style (inside HTML element)

Therefore, an inline style (within an HTML element) has the highest priority, which means it takes precedence over the style declarations in the <head>tag, style declarations in external style sheets, or in Style declaration (default) of the browser.
